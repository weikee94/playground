Headless UI — Modal

Framework-agnostic modal with focus trapping, keyboard navigation, and ARIA attributes.

Features

  • Escape to close
  • Tab / Shift+Tab focus trapping
  • Backdrop click to close
  • Body scroll lock
  • ARIA dialog attributes
  • Focus restoration to trigger

Example Modal

Try pressing Tab to cycle through focusable elements, Escape to close, or click the backdrop.
